These numbers are arranged such that the value following the "." is 10 times smaller than the value before the ".". To calculate the final score these values are added. This therefore means if the value to the right of the "." exceeds 9, the value to the left of the "." increments by one, additionally the amount by which this sum exceeds 9 will remain to the right of the ".". Finally if the value to the left of the "." exceeds 9, another number will be used to the right this value, these will not be separated by a "." but the value will nonetheless represent 10 of the values to the right of it.
